DOM(Document Object Model) - onload Event Listener

DOM 객체 : document 객체를 기반으로 생성되는 DOM 트리(Tree)의 자바스크립트 객체 - Node 객체(Element 객체)

브라우저로 웹프로그램을 요청하여 실행결과를 응답받아 브라우저(document 객체)에 출력

브라우저는 브라우저 엔진으로 응답받은 HTML 문서를 읽어 DOM Parser를 사용해 HTML 문서의 태그를 DOM 트리(Tree)로 생성하여 CSS 스타일을 적용해 브라우저에 출력 처리

자바스크립트를 사용해 태그에서 이벤트가 발생할 경우 DOM 트리에 존재하는 태그(DOM 객체)를 조작하여 동적인 페이지 구현 가능 - DHTML(Dynamic HTML)

반드시 DOM 트리가 완성돼야 동적인 페이지 구현 가능 - onload 이벤트 리스너 속성 사용
