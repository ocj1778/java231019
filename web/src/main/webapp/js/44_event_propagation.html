이벤트(Event) - 이벤트 전달(Event Propagation)

브라우저에서 발생된 이벤트는 window 객체에 먼저 도착하고 DOM 트리에서 이벤트가 발생된 타겟 객체(태그)로 이벤트가 이동되어 도착한 후 다시 반대 방향으로 이동해 window 객체로 되돌아오면 이벤트 소멸

이벤트가 이동되는 과정을 캡쳐링 단계와 버블링 단계로 구분

캡쳐링(Capturing) : window 객체로부터 이벤트가 발생된 타겟 객체까지 모든 객체에 이벤트가 전달되는 과정 - 이벤트가 전달된 객체에 등록된 이벤트 처리 함수 호출

버블링(Bubbling) : 이벤트가 발생된 타겟 객체부터 window 객체까지 모든 객체에 이벤트가 전달되는 과정 - 이벤트가 전달된 객체에 등록된 이벤트 처리 함수 호출
