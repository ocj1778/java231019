이벤트(Event) - 이벤트 처리

이벤트 : 태그(박스모델 - Element 객체)에서 발생되는 다양한 사건

이벤트 처리(Evnet Handle) : 태그에서 이벤트가 발생된 경우 자바스크립트로 작성된 이벤트 처리 명령 실행

이벤트 처리 방법-1 : 태그의 이벤트 리스너 속성(onXXX)의 속성값으로 이벤트 처리 명령을 작성하여 실행 - 이벤트 처리 명령으로 함수(이벤트 처리 함수) 호출

이벤트 처리 방법-2 : DOM 트리에서 DOM 객체를 검색해 Element 객체를 반환받아 Element 객체의 이벤트 리스너 속성에 이벤트 처리 함수 저장 - 이벤트 처리 함수 등록과 동일

이벤트 처리 방법-3 : DOM 트리에서 DOM 객체를 검색해 Element 객체를 반환받아 Element 객체로 addEventListener() 메소드를 호출하여 이벤트 처리 함수 등록
